# Dev Compose

docker-compose development environment for PHP.

- Apache 2
- PHP 7.4 via php-fpm
- PHP 8.0 via php-fpm
- PHP 8.1 via php-fpm
- Mysql 8.0

```
git clone https://github.com/s3b4stian/dev-compose.git
```

Before all need to build the environment
```
./build.sh
```

And finally start all
```
./up.sh
```

To stop all containers
```
./down.sh
```
